Sobre o Projeto Biomaker

O Projeto Biomaker é uma combinação de inovação tecnológica e educação científica. Com o objetivo de transformar o ensino de biologia, o projeto utiliza tecnologias como Raspberry Pi, impressão 3D e sensores IoT para criar uma experiência de aprendizado envolvente e acessível. Através dessas ferramentas, os alunos têm a oportunidade de explorar e entender a biologia de uma forma prática e imersiva.

O projeto foi desenvolvido em colaboração com escolas e universidades, proporcionando um ambiente de aprendizado colaborativ
